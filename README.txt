== Description ==

Admin-Todo is a todo list plugin for administrators to efficiently manage their tasks within the WordPress admin panel. With Admin-Todo, you can easily create, track, and prioritize tasks, ensuring that important items are not overlooked.

= Features =

- Create and manage todo items within the WordPress admin panel.
- Assign titles, content, links, and priorities to each todo item.
- Automatically retrieve the link from the current page.
- Sort todos based on priority for easy identification.
- Delete completed tasks with a single click.

== Installation ==

1. Upload the 'admin-todo' folder to the '/wp-content/plugins/' directory or install the plugin directly from the WordPress plugin directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the "Todo" menu item in the admin panel to start managing your tasks.

== Frequently Asked Questions ==

= Can I customize the priority levels? =

Yes, you can customize the priority levels to suit your needs. The plugin provides a default set of priority levels (high, medium, low), but you can modify them by editing the plugin code.

= How can I access the todo list? =

Once the plugin is activated, you can access the todo list by clicking on the "Todo" menu item in the WordPress admin panel.

= Can I delete completed tasks? =

Yes, you can delete completed tasks by clicking on the delete button next to each task in the todo list.
